============

This library contains simple (yet useful) controls for viewing and editing data, such as:

- **DatePicker**
- **DateTimePicker**
- **TimeSpan**
- **BindableRichTextBox**
- **DropDownButton**
- **FilterBox**
- **LogViewer**
- **WatermarkTextBox**
- **LinkLabel**
- **OpenFilePicker**
- **DirectoryPicker**
- **SaveFilePicker**

For documentation, please visit the [documentation portal](https://opensource.wildgums.com)
